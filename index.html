Work Experience

Greater Good in Education - Summer 2021

While working as a research apprentice for GGIE at UC Berkeley under Dr. Valerie Shapiro, I analyzed the use of online GGIE resources by 3000+ California educators using the SciPy ecosystem and webscraping tools like Beautiful Soup. In addition, I created Bash and Python scripts to automate the cleaning of SQL database files and creation of aggregated data flatmaps.

Lam Research Corporation - Summer 2020

At Lam, I designed and documented a flexible macro editor application for semiconductor wafer processing equipment using an Angular CLI and Electron.js framework. Additionally, I completed Lam's one-week Level II ML/AI Scientific Python course (included NumPy, pandas, Matplotlib, scikit-learn, and Yellowbrick).

Project: Website - Fall 2019

At Project: Website, I recruited non-profit organizations from 25+ countries seeking new websites through email and phone. Additionally, I served as a project manager, facilitating communcation and organizing workflow between website programmers and NPOs to ensure the success of each project.

Maskil - Summer 2019

At Maskil, I worked on the Django backend to create a "teams page" for each church organization from a MySQL database. Additionally, I created an admin permissions tool in order to manage specific songsheet creation features among Maskil editors.

Personal Projects

Collision Simulator (Java) - Spring 2019

For my senior AP CSP project, I programmed a collision simulator in Java that simulates collisions between two projectiles within a confined environment given initial mass, velocity, and position values. The simulator displays projectiles colliding in real-time with their current velocity and position values using JComponents.

Source Code

Chat Room (C) - Summer 2018

In CS 61C, I programmed a chat room system that supported message exchanges between users on separate console displays through the C socket interface. Additionally, the software supported slash-command functionality (\command), allowing users to mute other users and give them nicknames.

Send me an email if you'd like to see the source code.

Trailblazer (C++) - Summer 2017

At a Stanford Summer session, I took CS 106B and created Trailblazer, an application that takes two nodes and finds a path between them. In this project, I implemented 3 pathfinding algorithms (breadth-first, Djikstra's, and A*) with a visual display of each visited node and the final path result.

Send me an email if you'd like to see the source code.
